# Python Prototype

A quick prototype in Python to assess the feasibility of our Unity project:

- validate an MRP3SAT instance
- find a planar drawing given a valid MRP3SAT instance
- various transformations from one NP-Complete problem to another
- final drawing to be mapped to a BTD6 round

*Reminder:* to activate the venv, run `source venv/bin/activate`.
